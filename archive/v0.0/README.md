# v0.0 Archive

These files are the original v0.0 RAG prototype (ChromaDB + OpenAI + Streamlit).
They are preserved for reference but are not part of the v0.1 build.

To run v0.0, install the separate dependencies:
```
pip install chromadb openai streamlit python-dotenv
```
And set `OPENAI_API_KEY` in your `.env`.
